# [ItemEra] Expansion, Reagents, Profession, mats Tooltip & Filter (Bagnon / Baganator support) & SmexyMats alternative (outdated)

```
Filter your bags, bank, guild bank, and reagent bank by expansion to find specific items instantly. The tooltip displays the professions associated with each reagent, similar to SmexyMats (outdated).
```

**ItemEra** adds **expansion information to all item tooltips** with color-coded names and visual icons for instant recognition. Works seamlessly across inventory, bank, guild bank, and merchant interfaces, making it easy to identify item origins at a glance.

## **Features**

- Expansion info on item tooltips with icons and color-coded text
- **Crafting reagent profession display** - Shows which professions use each reagent
- Filter inventory, bank, and guild bank by expansion
- Full integration with **Baganator** and **Bagnon** bag addons
- Search query support for bag addon filtering
- Configurable tooltip display (show/hide name, logo, expansion text)
- Configurable profession display (icon + name, name only, or icon only)
- Modifier key support (show expansion info only when holding a key)
- Multi-language support (EN, ES, FR, IT, PT, RU, ZH)

## **Bag Addon Integration**

ItemEra integrates seamlessly with popular bag addons:

| Addon         | Features                                            |
| ------------- | --------------------------------------------------- |
| **Baganator** | Expansion filter dropdown with search query support |
| **Bagnon**    | Expansion filter dropdown with search query support |

When using these addons, ItemEra adds a dropdown selector to filter items by expansion using the addon's native search system.

## **Configuration**

Access settings via `/itemera` or through the AddOns settings panel:

**Expansion Display:**

- Show/hide expansion name
- Show/hide expansion logo
- Show/hide "Expansion:" text prefix
- Configure modifier key (Shift, Ctrl, Alt) to show expansion info

**Profession Display:**

- Show/hide profession name
- Show/hide profession icon
- Show/hide "Profession:" text prefix
- Enable/disable text wrapping for long profession lists

**Filter Options:**

- Enable/disable filters for inventory, bank, and guild bank
- Enable/disable Baganator integration
- Enable/disable Bagnon integration
